## Content SEO Checklist

### On-Page Optimization
- [ ] Every article optimized for 1-2 primary keywords
- [ ] Title tag includes primary keyword + benefit
- [ ] Meta description: 150-160 chars, includes target keyword
- [ ] H1 tag matches article intent (guide, comparison, tutorial)
- [ ] Internal linking: 2-5 relevant internal links per article
- [ ] Image optimization: alt tags, file size, WebP format

### Content Structure
- [ ] Word count: 1500-2500 words for deep-dive articles
- [ ] Scannable: H2/H3 headings, bullet points, code blocks
- [ ] CTA included: Every article has relevant next step (signup, docs, try demo)
- [ ] Schema markup: Article schema for SERP rich snippets (FAQ, HowTo)

### Technical SEO
- [ ] Page speed: < 2s load time (optimize images, defer non-critical JS)
- [ ] Mobile responsive: Test on multiple screen sizes
- [ ] Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- [ ] Sitemap: New content pages auto-added to sitemap.xml within 24h

## Content Approval Workflow

### QA Process
1. **Technical accuracy review:** Each article reviewed by technical SME before publishing
2. **SEO checklist verification:** SEO checklist completed and confirmed
3. **Link audit:** All internal and external links verified working
4. **Proofreading:** Grammar, spelling, formatting consistency
5. **Publication:** Approved content published across all channels with UTM tracking

### Content Calendar Management
- **Monthly review:** Last week of month, evaluate pillar performance
- **Monthly planning:** First week of month, plan next month's content
- **Keyword refinement:** Quarterly, adjust target keywords based on GSC performance
- **Backlink audit:** Quarterly, identify broken links, outreach opportunities

## Contingency Plans

### If organic growth < target (Month 1)
- **Accelerate content:** Increase frequency from 1x/week to 2x/week for 4 weeks
- **Aggressive backlink outreach:** Double outreach targets for 2 weeks
- **Paid amplification:** Request CMO approval for promoted content on niche communities

### If content ROI < target (Month 2)
- **Pivot content types:** Test video content (YouTube tutorials, technical demos)
- **Community engagement:** Host Reddit AMAs, Discord events, Twitter Spaces
- **Partnership outreach:** Explore integration partnerships for co-marketing

### If technical SEO issues arise
- **Immediate triage:** Diagnose within 24h, fix within 72h
- **Documentation update:** Record issue and resolution in internal knowledge base
- **Tool evaluation:** If persistent technical SEO issues, evaluate dedicated SEO tool (Screaming Frog, Ahrefs)
