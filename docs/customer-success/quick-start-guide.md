# Quick Start Guide - Faintech Lab Beta

**Time to Complete:** 5 Minutes
**Last Updated:** 2026-03-21

---

## Welcome to Agent Memory Core! 🧠

This guide gets you up and running in 5 minutes. Let's build your first AI agent with long-term memory.

---

## Step 1: Sign Up (1 min)

1. Go to the beta access link from your invitation email
2. Click "Create Account"
3. Enter your email and create a password
4. Verify your email (check your inbox)

✅ **Done:** You're in!

---

## Step 2: Create Your First Agent (2 min)

1. Click "Create New Agent"
2. Give it a name (e.g., "Research Assistant" or "Learning Bot")
3. Describe its purpose in one sentence:
   - Example: "Helps me remember research findings across projects"
4. Click "Create Agent"

✅ **Done:** Your agent exists!

---

## Step 3: Add Your First Memory (1 min)

1. Click your new agent
2. Find "Add Memory" or "+ New Memory"
3. Paste something you want your agent to remember:
   - Example: "Project X deadline is March 30th"
   - Example: "User prefers Python over JavaScript for data work"
4. Click "Save Memory"

✅ **Done:** Your agent now knows something!

---

## Step 4: Ask Your Agent a Question (1 min)

1. In the chat box, ask something related to what you just saved:
   - Example: "When is Project X due?"
   - Example: "What's my preferred language for data work?"
2. Watch your agent respond using its memory

✅ **Done:** You've seen Agent Memory Core in action!

---

## What's Next?
