## Emergency Escalation Paths

### If Distribution Platform Fails
- **Scenario:** HN auto-mod, Twitter rate limit, LinkedIn shadowban
- **Action:** Escalate to CEO within 1h
- **Backup:** Switch to alternative channels (Reddit, IndieHackers, Discord outreach)

### If Technical Issue Blocks Signups
- **Scenario:** Signup flow broken, 500 errors, payment gateway down
- **Action:** Alert CTO + DevOps immediately
- **Pause Distribution:** Hold all new posts until fixed
- **Public Communication:** Transparency about issue, not silence

### If Negative Feedback Spike
- **Scenario:** 20+ negative comments, 0 constructive engagement
- **Action:** Pause distribution, escalate to CEO
- **Analysis:** Root cause (messaging mismatch vs actual PMF issue)
- **Decision:** Pivot strategy or continue with transparency
