## Metrics to Track (Per Channel)

### Channel 1: GitHub Issue #90
- **Metric 1:** Views count
- **Metric 2:** Stars count
- **Metric 3:** Forks count
- **Success Threshold:** >100 stars = success trajectory

### Channel 2: Twitter/X
- **Metric 1:** Impressions
- **Metric 2:** Clicks
- **Metric 3:** Replies/Conversations
- **Success Threshold:** 200+ impressions, 20+ conversations

### Channel 3: Discord
- **Metric 1:** Reactions to announcement
- **Metric 2:** New members joined
- **Metric 3:** Questions asked in #general or #support

### Channel 4: LinkedIn
- **Metric 1:** Impressions
- **Metric 2:** Likes
- **Metric 3:** Comments
- **Success Threshold:** 500+ impressions, 15+ conversations

### Channel 5: Hacker News (Mar 26)
- **Metric 1:** Upvotes
- **Metric 2:** Comments
- **Metric 3:** Signups from HN traffic
- **Success Threshold:** >100 upvotes (success), >300 (viral), 50+ signups

## Conversion Funnel Metrics

### Funnel Stage 1: Traffic Sources
- Track: Which channels drive visitors?
- Record: Hourly breakdown for first 6 hours

### Funnel Stage 2: Landing Page Visits
- URL: https://faintech-lab.com (or actual landing page)
- Track: Unique visitors vs repeat visits

### Funnel Stage 3: Signups
- Track: Total signups by channel
- Record: Conversion rate (visits → signups)

### Funnel Stage 4: First Memory Creation (TTFV)
- Track: Users who create first memory
- Record: Time from signup to first memory (Time-To-First-Value)

### Funnel Stage 5: Active Users (Day 2+)
- Track: Users who return on Day 2+
- Record: Retention metric
