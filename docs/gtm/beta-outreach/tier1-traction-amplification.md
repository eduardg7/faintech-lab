## Amplification Triggers

### Trigger 1: First Positive Response (Day 1-2)
**Signal:** User responds positively to outreach, schedules demo or signs up
**Action:**
1. Send personalized onboarding email with 3 quick wins (session creation, memory write, team sync)
2. Offer 15-min onboarding call (optional)
3. Track activation: time-to-first-session

### Trigger 2: First Session Created (Day 2-3)
**Signal:** User creates first session with ≥2 agents
**Action:**
1. Send activation check-in: "First session live - what's your first goal?"
2. Capture early win metric: agents spawned, tasks completed
3. Ask permission to quote in HN launch

### Trigger 3: First Use Case Emerges (Day 3-4)
**Signal:** User demonstrates specific workflow (e.g., "using for CRM sync")
**Action:**
1. Deep-dive interview: 15-min call, focus on before/after
2. Document as case study draft (Problem → Faintech → Impact)
3. Offer co-created testimonial (user drafts + we polish)

### Trigger 4: Negative or Neutral Feedback (Any Day)
**Signal:** User reports bug, UX friction, or "not for me"
**Action:**
1. Immediate triage to CPO (if product) or CISO (if security)
2. Send acknowledgment: "Heard you - here's the fix timeline"
3. Close loop: "Fixed in v1.0.3 - please retry"
4. Convert blocker → trust signal (we ship fast)
