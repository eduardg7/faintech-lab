## Executive Summary

Mem0's 5x price increase ($49→$249/month for Pro tier) creates a narrow acquisition window for AMC to capture developers priced out of premium AI memory solutions. AMC's positioning as "80% cheaper than Mem0 Pro, EU data residency included" addresses both cost sensitivity and GDPR compliance concerns.

**Key Insight:** Mem0 gates graph features (entity relationships, multi-hop queries, structured knowledge) behind $249/month Pro tier. Standard tier ($19/month) provides vector search only. This creates a "price cliff" where teams must jump from $19 to $249 to access advanced features.
