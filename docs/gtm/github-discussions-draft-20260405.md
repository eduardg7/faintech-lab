### Title: Show HN: I built an AI that remembers your conversations across sessions

Hey everyone,

I've been working on **AMC (AI Memory Companion)** - a system that gives AI agents persistent memory across sessions.

**The problem:** Every time you start a new chat with an AI, it's like meeting a stranger. No context, no history, no relationship building.

**What I built:** A memory layer that:
- Stores conversation history with semantic search
- Extracts key facts automatically (preferences, projects, relationships)
- Makes memories available across all your AI sessions
- Works with local LLMs (privacy-first, no cloud dependency)

**Current status:** Beta launched this week, looking for early adopters to test the memory extraction quality.

**Demo:** https://amc-frontend-weld.vercel.app

**Tech stack:** Next.js, local vector DB, OpenAI API (optional), self-hostable

**Ask:** I'd love feedback on:
1. Is persistent AI memory something you've wanted?
2. What would you want the AI to remember about you?
3. Any deal-breakers for adoption (privacy, setup complexity, cost)?

Happy to answer questions about the architecture or implementation details.
