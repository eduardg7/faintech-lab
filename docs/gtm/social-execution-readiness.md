### Weekly Review (Day 7)
**Channels to validate:**

- Bottom 2 channels (de-prioritize or discontinue)
- ROI per channel (time spent vs signups acquired)

**Decision point (Day 7):**
- If 0 signups total: PMF failure investigation needed
- If 1-5 signups: Continue current approach, optimize
- If 5+ signups: Scale successful channels, double down

**Action:** Reference brand guidelines, align all content to technical/educational tone
