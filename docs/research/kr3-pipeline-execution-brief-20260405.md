## AC4: Pipeline Tracking System

### Stages
1. **Identified** (5 companies) - Current state
2. **Contacted** - First outreach sent
3. **Engaged** - Response received (any channel)
4. **Demo Scheduled** - Calendar invite sent
5. **Demo Completed** - Discovery/technical call done
6. **Pilot Proposed** - Formal pilot offer sent
7. **Pilot Accepted** - Contract signed, onboarding starts
8. **Pilot Active** - 6-8 week engagement
9. **Pilot Complete** - Success metrics evaluated
10. **Converted** - Paid subscription started

### Criteria for Stage Advancement
- **Contacted → Engaged:** Any response (email reply, LinkedIn accept, demo request)
- **Engaged → Demo Scheduled:** Calendar invite confirmed
- **Demo Completed → Pilot Proposed:** CTO/VP Engineering attended demo
- **Pilot Proposed → Pilot Accepted:** Legal/procurement approval
- **Pilot Active → Pilot Complete:** 6-8 weeks elapsed + metrics collected
- **Pilot Complete → Converted:** 4/5 success metrics met

### Metrics Dashboard (weekly tracking)
- **Pipeline Velocity:** Days per stage
- **Conversion Rate:** % advancing to next stage
- **Response Rate:** % of outreach generating engagement
- **Demo Rate:** % of engaged contacts scheduling demos
- **Pilot Acceptance Rate:** % of proposals accepted
- **Revenue Pipeline:** € value by stage (Startup: €299, Growth: €799, Enterprise: custom)
