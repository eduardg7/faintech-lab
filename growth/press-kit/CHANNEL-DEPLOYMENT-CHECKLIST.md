## Contingency Planning

### Scenario: Low Sign-up Rate (< 10 applications by Day 3)

**Actions:**
1. Analyze conversion funnel: Traffic → Sign-up → Approval
2. Audit landing page copy and CTAs
3. A/B test messaging (problem vs solution angle)
4. Increase press outreach volume
5. Leverage personal networks for referrals

### Scenario: High Sign-up Rate (> 50 applications by Day 3)

**Actions:**
1. Cap new applications temporarily
2. Prioritize technical fit over speed
3. Communicate waitlist status transparently
4. Scale support resources
5. Prepare to expand beta capacity

### Scenario: Negative HN Feedback

**Actions:**
1. Monitor sentiment, respond to technical concerns
2. Don't engage with trolls or non-constructive criticism
3. Document valid feedback for roadmap
4. Pivot messaging if fundamental misunderstanding
5. Learn from failure for next channel experiment
