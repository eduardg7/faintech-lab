# AMC Demo Video Script (2-Minute Walkthrough)

**Product:** Agent Memory Cloud (AMC) MVP
**Target Audience:** Technical founders, AI/LLC practitioners
**Tone:** Founder-to-founder, honest, technical
**Duration:** ~2 minutes

---

## Video Structure

### Hook (0:00-0:15)
**Visual:** Quick montage of AI agents working (code snippets, API calls, memory storage)
**Narration:** "Building agents that remember. Not just tokens that disappear when the conversation ends. Persistent memory that scales across thousands of sessions."

### Problem (0:15-0:30)
**Visual:** Split screen - left: messy notebooks/DB queries, right: clean API call
**Narration:** "You've seen this before. Agents write to SQLite, query vector databases, hack together context windows. It works. Until it doesn't. Scaling means reinventing the wheel every time."

### Solution (0:30-1:15)
**Visual:** Screen recording of AMC dashboard - showing:
1. Agent creating a memory (API call or dashboard)
2. Memory appearing in agent's timeline
3. Semantic search retrieving related memories
4. Cross-project memory access
**Narration:** "Agent Memory Cloud gives your fleet a shared brain. Write once, recall instantly. Your agents remember conversations across sessions, share insights across projects, and stay coherent as they scale."

### Key Features (1:15-1:40)
**Visual:** Quick cuts of:
1. Python SDK: `memory.write(...)` in action
2. TypeScript SDK: `@agentmemory/sdk` import
3. API response: JSON with embeddings and metadata
**Narration:** "Python SDK. TypeScript SDK. REST API with semantic search. Works with any LLM. No vendor lock-in."

### Call to Action (1:40-2:00)
**Visual:** Landing page with "Join Beta" button, then scrolling to pricing (beta = free during early access)
**Narration:** "We're launching beta March 24th. Early access is free. Ship agents that remember everything. Sign up at agentmemory.cloud."

## Production Notes

**Tools:**
- Video recording: CleanScreen X or Loom (4K, 60fps)
- Editing: Descript or Premiere Pro
- Voiceover: Professional voice talent (warm, slightly British tone)

**Style:**
- Clean, minimal UI (hide browser tabs, notifications)
- Cursor highlights: Yellow circle (visible on dark backgrounds)
- Text overlays: Segoe UI or system font, 24-32px
- Transitions: Quick cuts, no fade effects

**Music:**
- Intro: Upbeat tech electronic (15s)
- Body: Subtle ambient background (low volume)
- Outro: Same as intro, fade out
