# Changelog

All notable changes to the Agent Memory Cloud Python SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.1.0] - 2026-03-11

### Added
- Initial release of Agent Memory Cloud Python SDK
- Full API coverage for memory, agent, and project management
- Type-safe models using Pydantic
- Keyword and semantic search support
- Comprehensive error handling
- Automatic retry logic for rate limiting
- Support for Python 3.8+
- Full type hints for mypy
- Examples and documentation
