# Changelog

All notable changes to the Agent Memory Cloud TypeScript SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-11

### Added
- Initial release of Agent Memory Cloud TypeScript SDK
- Full API coverage for memory, agent, and project management
- Full TypeScript type definitions
- Keyword and semantic search support
- Comprehensive error handling
- Automatic retry logic for rate limiting
- Support for Node.js 16+
- ESM and CommonJS builds
- Examples and documentation
